# gatsby-tutorial

### Setting up Gatsby

1. Install Gatsby globally

```
npm install -g gatsby-cli
```

2. Create site
```
gatsby new [my_site]
```

3. Open directory and run gatsby with the following commands:

```
cd my_site
gatsby develop
```

4. The project will start in http://localhost:8000/
